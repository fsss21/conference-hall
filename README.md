# Conference Hall

React приложение с роутингом на JavaScript.

## Технологии

- React 18
- React Router DOM 6
- Vite

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Структура проекта

```
src/
  ├── components/     # Переиспользуемые компоненты
  │   └── Layout.js   # Компонент макета с навигацией
  ├── pages/          # Страницы приложения
  │   ├── Home.js     # Главная страница
  │   ├── About.js    # Страница "О нас"
  │   └── NotFound.js # Страница 404
  ├── App.js          # Главный компонент с роутингом
  ├── main.jsx        # Точка входа
  └── index.css       # Глобальные стили
```

## Роутинг

Приложение использует React Router DOM для навигации:

- `/` - Главная страница
- `/about` - Страница "О нас"
- `*` - Страница 404 для несуществующих маршрутов
